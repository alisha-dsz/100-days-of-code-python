# Day 3 - Treasure Island

## 📖 Overview

**Treasure Island** is a text-based adventure game where players make choices to navigate through different scenarios in search of hidden treasure. Each decision leads to a different outcome, making it an engaging introduction to conditional logic in Python.

---

## 🎯 Objective

Create an interactive adventure game that:

* Presents the player with a series of choices.
* Uses conditional statements to determine the outcome.
* Displays different endings based on the player's decisions.

---

## 🛠️ Concepts Practiced

* Conditional Statements (`if`, `elif`, `else`)
* Nested Conditions
* User Input (`input()`)
* String Methods (`lower()`)
* Logical Decision Making
* Print Statements
