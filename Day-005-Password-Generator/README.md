# Day 5 - Password Generator

## 📖 Overview

The **Password Generator** is a Python program that creates secure, random passwords based on the user's preferences. Users can specify the number of letters, symbols, and numbers to include, and the program generates a randomized password.

This project focuses on working with loops, lists, and the `random` module to build a practical utility.

---

## 🎯 Objective

Create a program that:

* Accepts the desired number of letters, symbols, and numbers.
* Randomly selects characters from predefined lists.
* Shuffles the selected characters.
* Generates a secure and randomized password.

---

## 🛠️ Concepts Practiced

* Lists
* `for` Loops
* Random Module (`random`)
* List Operations
* String Manipulation
* User Input (`input()`)
