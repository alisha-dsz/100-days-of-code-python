# Day 6 - Escaping the Maze

## 📖 Overview

**Escaping the Maze** is a maze-solving challenge completed in **Reeborg's World**, where a robot must navigate through a maze and reach the exit using Python programming.

This project focuses on developing problem-solving skills by implementing movement functions, loops, and conditional logic to create an algorithm that can solve different maze layouts.

---

## 🎯 Objective

Create a Python program that:

* Controls the robot's movement through the maze.
* Detects walls and available paths.
* Navigates from the starting point to the exit.
* Successfully solves different maze configurations.

---

## 🛠️ Concepts Practiced

* Functions
* `while` Loops
* Conditional Statements (`if`, `elif`, `else`)
* Logical Operators
* Algorithmic Thinking
* Problem Solving

---


> **Note:** This project is designed to run in the **Reeborg's World** online environment rather than as a standalone Python script.

## ▶️ How to Run

1. Open the **Reeborg's World** website.
2. Select the **Maze** challenge.
3. Copy the code from `solution.py`.
4. Paste it into the editor.
5. Run the program to watch the robot solve the maze.

## 💻 Expected Result

The robot successfully navigates through the maze by:

* Following the correct path.
