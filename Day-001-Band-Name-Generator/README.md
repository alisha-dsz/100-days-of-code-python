# Day 1 - Band Name Generator

## 📖 Overview

The **Band Name Generator** is a beginner-friendly Python project that generates a fun band name by combining the user's hometown with their pet's name.

This project introduces the fundamentals of Python programming, including working with variables, user input, and string manipulation.

---

## 🎯 Objective

Create a program that:

* Accepts the user's city of birth.
* Accepts the user's pet's name.
* Combines both inputs to generate a unique band name.

---

## 🛠️ Concepts Practiced

* Variables
* User Input (`input()`)
* String Concatenation
* Output using `print()`

---

## 📂 Files

```text
Day-001-Band-Name-Generator/
└── main.py
```

---

## ▶️ How to Run

1. Clone this repository.
2. Navigate to the project folder.
3. Run the program:

```bash
python main.py
```
