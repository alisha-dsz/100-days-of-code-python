# Day 4 - Rock Paper Scissors

## 📖 Overview

**Rock Paper Scissors** is a classic command-line game where the player competes against the computer. The computer randomly selects a move, and the program determines the winner based on the game's rules.

This project introduces the use of lists, randomization, and conditional logic to create an interactive game.

---

## 🎯 Objective

Create a game that:

* Allows the user to choose **Rock**, **Paper**, or **Scissors**.
* Randomly generates the computer's choice.
* Compares both choices and determines the winner.
* Displays the selected moves and the game result.

---

## 🛠️ Concepts Practiced

* Lists
* User Input (`input()`)
* Conditional Statements (`if`, `elif`, `else`)
* Random Module (`random`)
* Indexing
* Game Logic
